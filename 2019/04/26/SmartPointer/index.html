C++：SmartPointer

发表于 2019-04-26 更新于 2022-08-13 分类于 zztttt ， programming

为何使用智能指针？

  1. 因为能帮我们管理内存，避免内存泄漏
  2. 能帮我们处理空悬指针（俗称野指针，即delete掉指针后，原来的指针指向了垃圾区域）的问题

智能指针的类型

  • shared_ptr: 多个智能指针指向同一块区域，该对象会在最后一个引用被销毁时释放。
  • unique_ptr: 每个对象只能有一个智能指针指向它，避免资源泄漏。
